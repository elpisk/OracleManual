03   리눅스 기본 구조와 파일시스템 

### 3.1 리눅스 계층 구조 

리눅스는 모든 것을 파일로 취급합니다(Everything is a File). 최상위 디렉터리 / (루트)에서 시작하는

 트리 구조를 가집니다. 

/                     ← 루트 디렉터리 
├── bin/              ← 기본 실행 파일 (ls, cp 등) 
├── sbin/             ← 시스템 관리 명령어 (root용) 
├── etc/              ← 설정 파일 
├── home/             ← 일반 사용자 홈 디렉터리 
│   └── oracle/       ← 사용자 oracle의 홈 
├── root/             ← root 사용자 홈 
├── var/              ← 가변 데이터 (로그, 스풀) 
├── tmp/              ← 임시 파일 
├── usr/              ← 사용자 프로그램 
├── lib/              ← 라이브러리 
├── dev/              ← 디바이스 파일 
├── proc/             ← 프로세스 정보 (가상 파일) 
└── mnt/              ← 마운트 포인트

### 3.3 절대경로 vs 상대경로 

절대경로 (Absolute Path) 

루트(/)에서 시작하는 전체 경로 

/home/student/documents/report.txt 

상대경로 (Relative Path) 

현재 디렉터리 기준 경로 

기호 

의미 

. 

.. 

~ 

현재 디렉터리 

상위(부모) 디렉터리 

현재 사용자의 홈 디렉터리 

~root 

root 사용자의 홈 디렉터리 

Tip:  명령어에서 ~ 는 /home/현재사용자 와 동일합니다. cd ~ 는 홈 디렉터리로 이동합니다.
