### 6.1 Vim 이란? 

Vim(Vi IMproved)은 Unix 전통의 vi 에디터를 개선한 터미널 기반 텍스트 에디터입니다. 마우스 없

이 키보드만으로 모든 작업이 가능하여, 서버 원격 접속(SSH) 환경에서 설정 파일을 편집할 때 실무 

표준으로 사용됩니다. 

Vim 의 주요 특징 

•  키보드 중심 — 마우스 없이 빠른 편집 가능 

•  모달 편집(Modal Editing) — 모드 전환으로 명령과 입력을 분리 

•  모든 Linux/Unix 서버에 기본 설치 (vi 호환) 

•  강력한 검색·치환·매크로 기능 

•  Oracle DB 설정파일(/etc/oratab, listener.ora 등) 편집에 필수 

Vim 설치 확인 

```bash
# Vim 버전 확인 
$ vim --version | head -2 
```
VIM - Vi IMproved 8.2 (2019 Dec 12)

```

```


### 6.2 파일 열기

```bash
# 기본 사용법 
$ vim 파일명 
```


```bash
# 예시 
$ vim /etc/hosts              # 기존 파일 열기 
$ vim newfile.txt             # 새 파일 생성 
$ vim file1.txt file2.txt     # 여러 파일 동시 열기 
$ vim +50 /etc/passwd         # 50번째 줄에서 열기 
$ vim +/oracle /etc/passwd    # "oracle" 검색 위치에서 열기 
```


### 6.3 Normal 모드 vs Insert 모드  (02:08) 

Vim의 가장 중요한 개념은 모드(Mode)입니다. Vim은 현재 어떤 모드인지에 따라 키보드 입력의 의

미가 완전히 달라집니다.

### 6.8 유용한 Vim 설정 (~/.vimrc) 

홈 디렉터리의 .vimrc 파일에 설정을 저장하면 Vim 시작 시 자동 적용됩니다. 

```bash
$ vim ~/.vimrc 
```


" 줄 번호 표시 
set number 

" 구문 강조 (Syntax Highlighting) 
syntax on 

" 검색 시 대소문자 무시 
set ignorecase 

" 검색어 실시간 하이라이트 
set hlsearch 
set incsearch 

" 탭을 스페이스 4칸으로 
set expandtab 
set tabstop=4 
set shiftwidth=4 

" 현재 줄 강조 
set cursorline 

" 파일 타입 감지 
filetype plugin indent on 

Tip:  Oracle DB 설정파일(listener.ora, tnsnames.ora 등)은 모두 텍스트 파일이므로 Vim으로 

직접 편집합니다. :set nu 로 줄 번호를 활성화하면 편리합니다.
